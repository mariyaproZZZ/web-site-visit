# 📚 Электронный табель посещаемости

Готовое веб-приложение для учёта посещаемости.

## 🚀 Функционал
- **Студент:** расписание, отметка на опросах, портфолио
- **Преподаватель:** группы, опросы, табель посещаемости
- **Уведомления** о событиях

## 🛠️ Технологии
React 18 • TypeScript • Vite • SCSS Modules

## 🔐 Тестовые данные
- **Студент:** `student1` / `student123`
- **Преподаватель:** `teacher` / `teacher123`

## 🚀 Запуск
```bash
npm install
npm run dev
